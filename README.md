Simple Starter Kit for Gulp / Scss / Harmon based Website
=========================================================

This is my starter kit for simple websites.

## Install

```git clone https://github.com/hansspiess/hs-starter.git <project-name>```

```cd <project-name>```

```npm install```

```gulp build```
